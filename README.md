## what is this
CrewX is a minecraft client made to completely own brazilian servers like MushMC, Hylex, and Kaizen. we focused on making bypasses that actually work and keeping the performance high so your game doesnt lag when you're playing.

## how to build
1. clone the repo:
   ```bash
   git clone https://github.com/4everdies/CrewX.git
   ```
2. build it using gradle:
   ```bash
   ./gradlew build
   ```
3. grab the generated `.jar` file and you're good to go.
